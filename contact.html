Frequently Asked Questions

Is this open source?
Yes! The project is open source and available on GitHub. Feel free to explore the code and contribute.
Can I use this for my own project?
Absolutely! This project is licensed under MIT, so you're free to use, modify, and distribute it for your own purposes.
What hardware do I need?
To build this project, you need three main parts: an ESP32 (for WiFi), an STM32 (for processing), and a 433 MHz radio transmitter (for wireless communication).
Does it work with other microcontrollers?
The system is designed for ESP32 and STM32, but the protocol can be adapted to work with other microcontrollers that support UART communication.
